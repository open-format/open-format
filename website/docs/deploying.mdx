# Deploying

Deploying is the process of creating a new instance of the Open Format contract on the blockchain.
Open Format is currently available on the Polygon Mainnet and Polygon Mumbai networks. 

### Creating NFT collections

The SDK makes it simple to quickly and easily create fully customisable NFTs.

:::tip Gotcha's
The metadata URL should return a JSON blob of data with the metadata for your NFT. For now, we only support metadata store on IPFS.
:::

### Deploy

The deploy function creates a new instance of the Open Format contract on chain.





<Tabs groupId="sdk">
<TabItem value="js" label="JavaScript">

```jsx
import { OpenFormatSDK } from '@simpleweb/open-format';
  
  const sdk = new OpenFormatSDK({
    network: 'mumbai',
    signer: '0x...',
  });

  await sdk.deploy({
    maxSupply: 100,
    mintingPrice: 0.01,
    name: "TICKET",
    symbol: "FESTIVAL",
    url: "ipfs://bafkreifxgd3n6uzlhtenlwycphesqlmwf7dmqqzwho4iz6r6wuwtqtneza"
  });

```
