# Deploying

There is only one function you need in order for you to be able to deploy an NFT, the deploy function.

:::tip Gotcha's
The metadata URL should return a JSON blob of data with the metadata for your NFT. For now, we only support metadata store on IPFS.
:::

<Tabs groupId="sdk">
<TabItem value="js" label="JavaScript">

```js
import { OpenFormatSDK } from '@simpleweb/open-format';

const sdk = new OpenFormatSDK({
  network: 'mumbai',
});

await sdk.deploy({
  maxSupply: 100,
  mintingPrice: 0.01,
  name: "TICKET",
  symbol: "FESTIVAL",
  url: "ipfs://bafkreifxgd3n6uzlhtenlwycphesqlmwf7dmqqzwho4iz6r6wuwtqtneza"
});
```

</TabItem>
<TabItem value="react" label="React">

```jsx
import { useDeploy, useWallet } from '@simpleweb/open-format-react';

function MyComponent() {
  const { isConnected } = useWallet();
  const { deploy } = useDeploy();

  return (
    <>
      {isConnected && (
        <button
          onClick={() =>
            deploy({
              maxSupply: 100,
              mintingPrice: 0.01,
              name: "TICKET",
              symbol: "FESTIVAL",
              url: "ipfs://bafkreifxgd3n6uzlhtenlwycphesqlmwf7dmqqzwho4iz6r6wuwtqtneza"
            })
          }
        >
          Deploy NFT
        </button>
      )}
    </>
  );
}
```
